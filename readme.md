# 8 Gallery
Art sharing site

## Usage
- server `http://localhost:3000`
Clone the project then:
1. For building and starting server 
```
cd 8-Gallery
npm run deploy
```
2. For starting dev client & serve
```
cd 8-Gallery
npm run dev
```
3. For building 
```
cd 8-Gallery
npm run build
```

## Feature
- File Upload
- mariadb bigIntAsNumber
- Node fs
- Dump err to "ErrorLogs (Date).log"

- use my `PHP` serve to store images
